About This Project

GlobalX Currency Converter is a lightweight, client-side web app for quick currency conversion. It uses static exchange rates (Nov 2025) and requires no internet connection to run.

  • 💡 Built with vanilla HTML, CSS, and JavaScript — zero dependencies
  • 🌙 Dark & light theme toggle
  • 📜 Conversion history with fade-in animations
  • 📱 Fully responsive (desktop & mobile)
  • ♿ Accessible markup with ARIA labels
